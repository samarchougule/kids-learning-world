# Kids Learning World - Updated Linked Package

## What is updated
- `marathi-alphabet.html` now uses Marathi-focused words only.
- `numbers.html` now shows the number and spelling on the left, and the same count of circular shapes on the right.
- `index.html` landing page is linked to all updated slideshow pages.

## How to run
1. Extract this ZIP.
2. Open `index.html` in Edge or Chrome.
3. Click any card.

## Deploy
Upload all files to GitHub Pages / Azure Static Web Apps / Cloudflare Pages / Netlify.


## Final voice fixes
- Marathi alphabet voice now says phrases like `अ म्हणजे अननस`, `आ म्हणजे आंबा`.
- Numbers voice now speaks only the number word once, for example `One`, `Two`, `Three`; the number digit is still shown on the left and circles are shown on the right.
